Добро пожаловать в документацию **API Contract Hub** — платформы для управления жизненным циклом OpenAPI-контрактов в микросервисных архитектурах.

:::tip Для кого эта документация
Документ предназначен для команды разработки, архитекторов, DevOps-инженеров и технических руководителей ООО «ПринтСтройМонтажСервис».
:::

## Ключевые метрики проекта

| Метрика | Исходный показатель | Цель |
|---------|-------------------|------|
| API-инциденты в месяц | 18 | ≤ 5,4 (−70%) |
| Время анализа влияния | 4 часа | ≤ 30 минут (−87,5%) |
| Покрытие микросервисов | 0% | 62% (75 из 120) |
| NPS пользователей | — | ≥ 50 |

## Технологический стек

**Серверная часть** — Node.js + Express.js  
**Клиентская часть** — React.js + React Flow  
**База данных** — PostgreSQL с JSONB  
**Развёртывание** — Docker Compose (MVP) → Kubernetes (production)
